# SeqUIaSCOPE Helm Chart

A Helm chart for deploying [SeqUIaSCOPE](https://github.com/katjur01/seqUIaSCOPE/tree/dev), a platform designed for routine clinical oncology diagnostics through patient-centric integration and visualization of genomic variants, fusion events, and expression profiles. The application supports embedded genome browsing, dynamic pathway visualization, and flexible reporting with customizable templates.
